# Práctica 2 - Conceptos Báscos de Base De Datos
 1. ¿Qué es una Base de Datos Relacional? 
Es una forma de almacenar información usando tablas que se relacionan entre sí para mantener los datos ordenados y evitar repeticiones, se usa mucho en sistemas, aplicaciones y páginas web porque facilita la organización y el manejo de grandes cantidades de información.

 2. ¿Qué es una Tabla en el contexto de Base de Datos Relacionales?
 Es como una hoja de Excel donde se guardan los datos, está formada por filas y columnas y cada tabla almacena información de un solo tema como usuarios, productos o registros específicos.

 3. ¿Qué es una relación en el contexto de Base de Datos Relacionales? 
Es la conexión que existe entre dos o más tablas para compartir información sin duplicarla, se logra usando claves y permite que los datos estén bien organizados y conectados entre sí.

 4. ¿Qué representa una columna en el contexto de Base de Datos Relacionales? 
Una columna representa una característica del dato que se va a guardar, define el tipo de información como nombre, edad o correo y todas las filas deben seguir ese mismo formato.

 5. ¿Qué representa un renglón en el contexto de Base de Datos Relacionales? 
Un renglón representa un registro completo dentro de la tabla, contiene toda la información de una persona u objeto y reúne los valores de cada columna.

 6. ¿Qué es un motor de base de datos relacional?
 Es el programa encargado de manejar la base de datos, permite guardar, consultar y modificar la información y también se encarga de ejecutar las instrucciones SQL para que todo funcione correctamente.

 7. ¿Cuáles son los motores de bases de datos relacionales más populares? 
Entre los más conocidos están MySQL, PostgreSQL, Oracle y SQL Server, todos se usan ampliamente dependiendo del tipo de sistema y las necesidades del proyecto.

 8. ¿Cuál es el motor de base de datos ligero que no necesita un servidor? 
SQLite es un motor ligero que funciona sin servidor, guarda toda la información en un solo archivo y es muy usado en aplicaciones pequeñas y móviles.

 9. ¿Qué significa SQL en el contexto de bases de datos relacionales? 
SQL significa Structured Query Language y es el lenguaje que se usa para trabajar con bases de datos relacionales, permite consultar, insertar, actualizar y eliminar información.

 10. ¿Cuáles son los elementos de un sistema de información basado en bases de datos relacionales? 
Incluye la base de datos, el motor que la administra, las aplicaciones que usan la información, los usuarios y las reglas que aseguran que los datos estén organizados y protegidos.
